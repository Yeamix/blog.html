Question-1 : What is The difference between local storage and session storage?

Answer : Local storage and session storage works in a same way.But local storage doesn't clears after closing window or turn of your pc etc. but session storage clears all.

Question-2 : What is the difference between global scope and block scope?

Answer : In a browser environment, the global scope is controlled by the window object while in Node. js, it's controlled by the global object. Block scopes are what you get when you use if statements, for statements, and the like.

Question-3 : Event Loop

Answer : The event loop is a process that waits for the Call Stack to be clear before pushing callbacks from the Task Queue to the Call Stack. Once the Stack is clear, the event loop triggers and checks the Task Queue for available callbacks. If there are any, it pushes it to the Call Stack, waits for the Call Stack to be clear again, and repeats the same process.
